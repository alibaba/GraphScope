Graph types
------------------

+----------------+------------+--------------------+------------------------+
|    Class       | Type       | Self-loops allowed | Parallel edges allowed |
+================+============+====================+========================+
| Graph          | undirected | Yes                | No                     |
+----------------+------------+--------------------+------------------------+
| DiGraph        | directed   | Yes                | No                     |
+----------------+------------+--------------------+------------------------+

Notice that graphscope.nx not support MultiGraph and MultiDiGraph.
