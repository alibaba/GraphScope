graphscope.framework.dag_utils.fetch_gremlin_result(result_set, fetch_type='one')[source]

Fetch the gremlin query result.

Parameters

  • result_set (raphscope.interactive.query.ResultSetDAGNode) – The instance holds the resultSet in coordinator that can fetch the gremlin result from.

  • fetch_type (str) – “one” or “all”. Defaults to “one”.

Returns

An op to fetch the gremlin result.
