graphscope.get_option¶

graphscope.get_option(key)[source]¶

Get the value of specified option.

Available options:

  • num_workers

  • log_level

  • show_log

  • vineyard_shared_mem

  • k8s_namespace

  • k8s_service_type

  • k8s_gs_image

  • k8s_etcd_image

  • k8s_gie_graph_manager_image

  • k8s_zookeeper_image

  • k8s_image_pull_policy

  • k8s_image_pull_secrets

  • k8s_coordinator_cpu

  • k8s_coordinator_mem

  • k8s_vineyard_daemonset

  • k8s_vineyard_cpu

  • k8s_vineyard_mem

  • k8s_engine_cpu

  • k8s_engine_mem

  • k8s_mars_worker_cpu

  • k8s_mars_worker_mem

  • k8s_mars_scheduler_cpu

  • k8s_mars_scheduler_mem

  • with_mars

  • k8s_waiting_for_delete

  • engine_params

  • initializing_interactive_engine

  • timeout_seconds

Parameters

key – str Key of GraphScope config you want to get.

Raises

ValueError – If no such option exists.

Returns: result: the value of the option
