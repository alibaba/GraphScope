快速开始NeuG Web UI

NeuG Web UI

NeuG Web UI 提供了一个基于浏览器的界面，用于与 NeuG 数据库进行交互。它提供了一种直观的方式来执行 Cypher 查询、可视化结果，并通过现代化的 Web 界面探索你的图数据。

基本用法

启动 Web UI

在启动 neug-cli 之后，启动 Web UI 的最简单方法是：

neug > :ui

这将启动打开数据库的 Web 服务器，地址为 http://127.0.0.1:5000。

自定义主机和端口

要在不同的主机或端口上运行：

neug > :ui 127.0.0.1:8080

示例

示例 1：使用本地数据库启动

 
# 使用特定数据库启动 UI
neug-cli open ./my_graph_db
neug > :ui

示例 2：公共访问

 
# 允许从任何 IP 地址访问
neug-cli open ./my_graph_db
neug > :ui 0.0.0.0:8080

Web 界面功能

Web UI 运行后，打开浏览器并导航到指定的 URL（默认：http://127.0.0.1:5000）。

查询界面

Web 界面提供：

  • 查询编辑器：编写和执行 Cypher 查询
  • 结果展示：以表格格式查看结果
  • 模式浏览器：浏览数据库模式
  • 查询历史：访问之前执行的查询

API 端点

Web UI 还暴露了 REST API 端点：

模式端点

GET /schema

以 JSON 格式返回数据库模式。

查询执行端点

POST /cypher
Content-Type: text/plain
 
{
    "query": "MATCH (n) RETURN n LIMIT 10",
    "format": "json"
}

执行 Cypher 查询并以 JSON 格式返回结果。

故障排除

端口已被占用

如果端口 5000 已被占用，请指定一个不同的端口：

neug > :ui localhost:8080

数据库连接问题

确保数据库目录存在且可访问：

 
# 检查目录是否存在
ls -la /path/to/your/database

以正确的权限启动

neug-cli open /path/to/your/database neug > :ui


### 缺少依赖项
如果看到导入错误，请安装所需的依赖项：
```bash
pip install flask flask-cors

安全注意事项

  • Web 界面默认在本地主机上运行以确保安全
  • 使用 0.0.0.0 时，请确保已设置适当的防火墙规则
  • 在生产环境中，考虑使用反向代理（nginx、Apache）
  • 生产环境中切勿使用调试模式

Web 界面默认从 CDN 加载资源。对于离线开发，你可能需要将静态资源下载到本地。
