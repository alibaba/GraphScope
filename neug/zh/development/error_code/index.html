开发指南错误代码

NeuG 错误码

NeuG 在 protobuf 文件 error.proto 中定义了所有运行时/服务错误码。该文件中的枚举被编译到每个组件中，因此开发者应将该文件中的定义视为唯一可信来源。下表总结了每个代码及其含义，以便在调试或显示错误时快速参考。

类别代码数值含义
通用OK0操作成功；无错误。
通用ERR_PERMISSION1001由于缺少权限而阻止操作。
通用ERR_VERSION_MISMATCHED1002二进制/数据库版本与数据目录不兼容。
通用ERR_DIRECTORY_NOT_EXIST1003目标目录路径不存在。
通用ERR_DATABASE_LOCKED1004数据目录被其他进程锁定。
通用ERR_DISK_SPACE_EXHAUSTED1005磁盘空间不足，无法继续。
通用ERR_CORRUPTION_DETECTED1006文件似乎已损坏或格式不符合预期。
通用ERR_INVALID_PATH1007提供的文件系统路径无效。
通用ERR_CONFIG_INVALID1008配置文件或值格式错误。
通用ERR_INVALID_ARGUMENT1009API/输入参数缺失或格式错误。
通用ERR_NOT_FOUND1010请求的资源（顶点、文件等）无法找到。
通用ERR_NOT_SUPPORTED1011当前上下文不支持该功能或操作。
通用ERR_INTERNAL_ERROR1012意外的内部故障；检查日志获取堆栈跟踪。
通用ERR_ILLEGAL_OPERATION1013当前状态/配置下不允许该操作。
通用ERR_IO_ERROR1014底层文件系统或设备 I/O 故障。
通用ERR_BAD_ENCODING1015遇到不支持的编码/解码。
通用ERR_INVALID_FILE1016引用的文件不存在或不可读。
通用ERR_EXTENSION1017加载或执行扩展时失败。
网络/会话ERR_NETWORK2001通用网络传输错误。
网络/会话ERR_SESSION_CLOSED2002会话句柄已关闭，不再可用。
网络/会话ERR_CONNECTION_CLOSED2003连接断开（服务器关闭或数据库关闭）。
网络/会话ERR_POOL_EXHAUSTED2004连接/会话池已用尽可用条目。
网络/会话ERR_SERVICE_UNAVAILABLE2005服务离线或尚未就绪。
网络/会话ERR_LOAD_OVERFLOW2006服务过载；客户端应重试/退避。
网络/会话ERR_CONNECTION_ERROR2007建立或维护连接失败。
查询编译与执行ERR_COMPILATION3000查询编译阶段失败。
查询编译与执行ERR_QUERY_EXECUTION3001执行查询计划时的通用运行时故障。
查询编译与执行ERR_QUERY_SYNTAX3002查询语法或语义无效。
查询编译与执行ERR_QUERY_TIMEOUT3003查询超过配置的执行时间限制。
查询编译与执行ERR_CONCURRENT_WRITE3004检测到冲突的并发写入。
查询编译与执行ERR_CODEGEN_ERROR3005查询代码生成期间失败。
查询编译与执行ERR_EMPTY_RESULT3006规划器推断结果集为空。
查询编译与执行ERR_NOT_INITIALIZED3007查询执行前数据库/会话未初始化。
事务与 WALERR_TX_STATE_CONFLICT4001事务状态冲突（如无效转换）。
事务与 WALERR_WAL_WRITE_FAIL4002无法追加到预写日志。
事务与 WALERR_TX_TIMEOUT4003事务超过超时限制。
Schema 与类型ERR_SCHEMA_MISMATCH5001操作与存储数据之间的 Schema 不匹配。
Schema 与类型ERR_INVALID_SCHEMA5002Schema 定义无效。
Schema 与类型ERR_TYPE_CONVERSION5003请求了非法的类型转换。
Schema 与类型ERR_TYPE_OVERFLOW5004值无法放入目标数据类型。
Schema 与类型ERR_INDEX_ERROR5005索引/偏移超出范围。
Schema 与类型ERR_PROPERTY_NOT_FOUND5006请求的属性在 schema 中缺失。
部署/平台ERR_PLATFORM_ABI6001二进制文件与主机平台之间的 ABI 不匹配。
部署/平台ERR_PY_BIND_INIT6002Python 绑定初始化失败。
部署/平台ERR_ARCH_MISMATCH6003二进制架构与运行时环境不匹配。
部署/平台ERR_DEPLOY_DEPENDENCY6004部署期间缺少运行时依赖。
功能缺口ERR_NOT_IMPLEMENTED7001功能占位符；尚未实现。
通用ERR_UNKNOWN9999未分类错误；检查日志获取上下文。

提示： 添加新错误码时，首先更新 proto/error.proto，必要时重新生成 protobuf 输出，并扩展此文档以便其他开发者快速发现新代码。
