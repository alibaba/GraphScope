连接对象

class Connection(object)

Connection 表示到数据库的逻辑连接。用户应使用此类与数据库进行交互， 例如执行查询和管理事务。 连接由 Database.connect 方法创建，不再需要时应通过调用 close 方法关闭。 如果数据库被关闭，所有到该数据库的连接将自动关闭。

is_open

@property

检查连接是否打开。

  • 返回:
    • bool 如果连接已打开则返回 True，否则返回 False。

execute

def execute(query: str,
            access_mode="",
            parameters: dict[str, Any] | None = None) -> QueryResult

在数据库上执行 Cypher 查询。用户可以在单个字符串中指定多个查询， 以分号分隔。查询将按照指定的顺序执行。 如果任何查询失败，整个执行将被回滚。 如果查询是 DDL 查询，例如 CREATE TABLE、DROP TABLE 等，数据库将相应地被修改。

有关查询语法的详细信息，请参阅 Cypher 手册文档。 查询的结果将以 QueryResult 对象的形式返回，该对象包含查询结果和查询元数据。 QueryResult 对象类似于迭代器，提供遍历结果的方法， 如 __iter__ 和 __next__。

如果查询是 DDL 或 DML 查询，结果将是空的 QueryResult 对象。

某些 Cypher 查询可以更改数据库状态，例如 CREATE TABLE、INSERT、 UPDATE、DELETE 等。其他查询，例如 MATCH(n) RETURN n.id，不会更改数据库状态， 但会返回查询结果。

如果数据库以只读模式打开，任何 DDL 或 DML 查询都会引发异常。 如果数据库以读写模式打开，所有查询都可以执行，并且数据库状态将相应更改。

 
    >>> from neug import Database
    >>> db = Database("/tmp/test.db", mode="w")
    >>> conn = db.connect()
    >>> res = conn.execute('CREATE TABLE person(id INT64, name STRING);')
    >>> res = conn.execute('CREATE TABLE knows(FROM person TO person, weight DOUBLE);')
    >>> res = conn.execute('COPY person FROM "person.csv"')
    >>> res = conn.execute('COPY knows FROM "knows.csv" (from="person", to="person");')
    >>> res = conn.execute('MATCH(n) RETURN n.id')
    >>> for record in res:
    >>>    print(record)
    >>> res = conn.execute('MATCH(p:person)-[knows]->(q:person) RETURN p.id, q.id LIMIT 10;')
    >>> # 提交带参数的查询
    >>> res = conn.execute(
        'MATCH (n:person) WHERE n.id = $id RETURN n.name', access_mode='r', parameters={'id': 12345})

  • 参数:

    • query (str) 要执行的查询。
    • access_mode (str) 查询的访问模式。可以是 read(r)、insert(i)、update(u)（包括删除）， 或用于模式修改的 schema(s)。用户应为查询指定正确的访问模式 以确保数据库的正确性。如果未指定访问模式，将默认设置为 update。支持的访问模式有：
      • read,r,READ,R: 用于只读查询
      • insert,i,INSERT,I: 用于仅插入查询
      • update,u,UPDATE,U: 用于更新查询（包括删除）
      • schema,s,SCHEMA,S: 用于模式修改操作
    • parameters (dict[str, Any] | None) 查询中要使用的参数。参数应该是一个字典，其中键是参数名称，值是参数值。如果不需要参数，可以设置为 None。

  • 返回:

    • query_result (QueryResult) 查询结果。
