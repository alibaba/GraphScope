数据导入导出加载数据

加载数据

LOAD FROM 是 NeuG 数据导入流水线的基础。它读取外部文件，自动推断 Schema（列名和类型），并生成仅在查询执行期间存在的临时结果集。无需预先定义 Schema。

你可以直接对加载的数据执行标准的关系操作——投影、过滤、类型转换、聚合、排序等。这使得 LOAD FROM 非常适合数据探索、验证和即席分析。

基本语法

LOAD FROM "<file_path>" (<options>)
[WHERE <condition>]
RETURN <column_list>
[ORDER BY <column> [ASC|DESC]]
[LIMIT <n>];

参数

  • <file_path> — 外部数据源的路径。目前仅支持本地文件系统路径。
  • <options> — 格式相关和性能相关的选项（见下文）。
  • RETURN <column_list> — 要返回的列。使用 * 返回所有列。

格式选项

CSV

CSV 是内置格式。以下选项控制 CSV 文件的解析方式：

选项类型默认值说明
delimchar|字段分隔符。可以是单个字符（如 ','）或转义字符（如 '\t'）
headerbooltrue第一行是否包含列名
quotechar"用于包围字段值的引号字符
escapechar\特殊字符的转义字符
quotingbooltrue是否启用引号处理
escapingbooltrue是否启用转义字符处理

示例：

LOAD FROM "person.csv" (delim=',', header=true)
RETURN name, age;

JSON / JSONL

JSON 支持通过 JSON 扩展 提供。安装并加载扩展后，LOAD FROM 可以读取 .json 和 .jsonl 文件：

INSTALL json;
LOAD EXTENSION json;
 
LOAD FROM "person.json"
RETURN *;

详见 JSON 扩展 页面了解格式特定选项和示例。

Parquet

Parquet 支持计划在 v0.2 版本中提供。

关系操作

LOAD FROM 支持对加载数据执行丰富的关系操作。以下所有示例均使用 Modern 数据集。

列投影与重排

列可以按任意顺序返回，不受源文件中列顺序的限制：

LOAD FROM "knows.csv" (delim=',')
RETURN weight, dst_name, src_name;

列别名

使用 AS 为列分配别名：

LOAD FROM "knows.csv" (delim=',')
RETURN src_name AS src, dst_name AS dst, weight AS score;

去重

使用 RETURN DISTINCT 从结果中移除重复行：

LOAD FROM "person.csv" (delim=',')
RETURN DISTINCT name;

你也可以对多列使用 DISTINCT：

LOAD FROM "person.csv" (delim=',')
RETURN DISTINCT name, age;

类型转换

使用 CAST 函数将列值转换为指定类型：

LOAD FROM "person.csv" (delim=',')
RETURN name, CAST(age, 'DOUBLE') AS double_age;

WHERE 过滤

使用 WHERE 子句过滤行。多个条件可以通过 AND、OR 和 NOT 组合：

LOAD FROM "person.csv" (delim=',')
WHERE age > 25 AND age < 40
RETURN name, age;

聚合

LOAD FROM 支持常用聚合函数（COUNT、SUM、AVG、MIN、MAX）和分组聚合：

LOAD FROM "person.csv" (delim=',')
RETURN
    COUNT(*) AS total,
    AVG(age) AS avg_age,
    MIN(age) AS min_age,
    MAX(age) AS max_age;
LOAD FROM "person.csv" (delim=',')
RETURN name, AVG(age) AS avg_age;

排序与限制

LOAD FROM "person.csv" (delim=',')
RETURN name, age
ORDER BY age DESC, name ASC
LIMIT 10;

性能选项

对于大文件，以下选项可以提升读取性能：

选项类型默认值说明
parallelboolfalse启用多线程并行读取（最多使用 CPU 核心数）。

注意： 批量读取选项（batch_read、batch_size）目前仅在 COPY FROM 中支持，不适用于 LOAD FROM。

示例：

LOAD FROM "large_person.csv" (
    delim = ',',
    header = true,
    parallel = true
)
RETURN name, age;
